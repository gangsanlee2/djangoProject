단축키 패턴 1: 컨트롤 + 소문자
    Ctrl + b : 메소드 선언부로 이동하기
    Ctrl + d : 라인 복제
    Ctrl + g :  라인번호로 이동
    Ctrl + n : 클래스명 검색
    Ctrl + w : 커서 위치 기준으로 블록 잡기. 반복할 수록 블록 확대
    Ctrl + y : 라인 삭제
    Ctrl + z : 실행취소

단축키 패턴 2: 컨트롤 + 대문자
    Ctrl + F : 동일 파일에서 글자 찾기
    Ctrl + Z : 재실행

단축키 패턴 3: 컨트롤 + 알트 + 소문자
    Ctrl + Alt + m : 메소드 추출 리팩토링
    Ctrl + Alt + o : 불필요한 임포트 삭제
    Ctrl + Alt + s : Setting 창 열기
    Ctrl + Alt + t : 선택된 블록 if, for, while 로 감싸기
    
단축키 패턴 4: 컨트롤 + 알트 + 대문자

단축키 패턴 5: 쉬프트
    Shift + F9 : 디버그 실행
    Shift + F10 : 실행
    더블 Shift : 파일 검색

단축키 패턴 6: 펑션키
    F3 : 검색한 것 다음으로 넘기기
    F9 : 다음 디버그로 넘김

단축키  패턴 7: 알트
    Alt + Enter : 임포트
    Alt + 화살표 : 메소드 단위로 커서 이동
